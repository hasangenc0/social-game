# 🎮 Game Project (under development)

### ⚡️ Start Go Game Server

```
cd server
make build-api
./bin/api
```

### 🏭 Build Web Assembly and JS

```
cd client
sh build sh
yarn run start /* For development build */
```

Go http://localhost:8080
